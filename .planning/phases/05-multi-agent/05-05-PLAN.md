       - 'use client' directive (zustand stores used in client components)

       - Interface AgentStatusUpdate { agentId: string; status: 'idle' | 'running' | 'error' }
       - Interface AgentChunkUpdate { agentId: string; chunk: string }
       - Interface AgentMessageUpdate { agentId: string; message: string; targetAgentId?: string }
       - Interface ActivityEvent { id: string; type: 'system_event' | 'agent_message' | 'tool_call' | 'conflict'; agentId?: string; targetAgentId?: string; content: string; timestamp: Date }
       - Interface WorkspaceStoreState:
         - connectionStatus: 'connected' | 'reconnecting' | 'disconnected'
         - agentStatuses: Map<string, 'idle' | 'running' | 'error'>
         - activityFeed: ActivityEvent[]
         - streamingChunks: Map<string, string> (agentId -> accumulated chunks)
         - setConnectionStatus(status)
         - updateAgentStatus(agentId, status)
         - addActivity(event)
         - appendChunk(agentId, chunk)
         - clearChunks(agentId)
         - reset()
       - Export const useWorkspaceStore = create<WorkspaceStoreState>((set, get) => ({ ... }))
       - activityFeed capped at 500 events (shift oldest when exceeded)

    2. Create `packages/web/lib/realtime/use-workspace-socket.ts`:
       - 'use client' directive
       - Import { useEffect, useRef } from 'react'
       - Import { io, type Socket } from 'socket.io-client'
       - Import { useWorkspaceStore } from './workspace-store'
       - Export function useWorkspaceSocket(tenantId: string, workspaceId: string):
         - const socketRef = useRef<Socket | null>(null)
         - const { setConnectionStatus, updateAgentStatus, addActivity, appendChunk, clearChunks } = useWorkspaceStore()
         - useEffect with [tenantId, workspaceId] deps:
           - const socket = io(process.env.NEXT_PUBLIC_WORKER_SOCKET_URL || 'http://localhost:3002', { reconnection: true, reconnectionAttempts: 5, reconnectionDelay: 1000 })
           - const joinRoom = () => socket.emit('workspace:join', tenantId, workspaceId)
           - socket.on('connect', () => { setConnectionStatus('connected'); joinRoom() })
           - socket.on('reconnect', () => { setConnectionStatus('connected'); joinRoom() }) // REAL-05
           - socket.on('disconnect', () => setConnectionStatus('disconnected'))
           - socket.io.on('reconnect_attempt', () => setConnectionStatus('reconnecting'))
           - socket.on('workspace:joined', () => { /* room joined confirmation */ })
           - socket.on('agent:status', (data) => updateAgentStatus(data.agentId, data.status))
           - socket.on('agent:message', (data) => addActivity({ id: crypto.randomUUID(), type: 'agent_message', agentId: data.agentId, targetAgentId: data.targetAgentId, content: data.message, timestamp: new Date() }))
           - socket.on('agent:chunk', (data) => appendChunk(data.agentId, data.chunk))
           - socket.on('agent:done', (data) => clearChunks(data.agentId))
           - socketRef.current = socket
           - return () => { socket.emit('workspace:leave', tenantId, workspaceId); socket.disconnect() }
         - Return socketRef

    3. Add Phase 5 animation variants to `packages/web/lib/motion.ts`:

       - Keep all existing variants untouched
