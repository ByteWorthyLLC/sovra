Quickstart

Get Sovra running in under 30 minutes.

Use this setup path to move from clone to a working multi tenant AI app with worker services and retrieval.

Step 1. Install and bootstrap

git clone https://github.com/ByteWorthyLLC/sovra.git
cd sovra
pnpm setup:local

Step 2. Start data and app services

supabase start
cd packages/web && pnpm dev
cd ../../packages/worker && go run ./cmd/worker
  • Fill Supabase keys in `.env.local` and `packages/web/.env.local`
  • Verify web health at `/api/health`
  • Verify worker health at `/health`

Quickstart FAQ

Do I need all integrations on day one?
No. Stripe, analytics, and advanced providers are optional until needed.
Can I deploy only the web app first?
Yes, but full agent and realtime workflows require the worker service.
Where is the production checklist?
Use `docs/production-readiness.md` and `docs/customer-onboarding-dx.md`.
